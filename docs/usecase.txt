Use Case UC-01 – Quản lý tài khoản & phân quyền
Use Case ID: UC-01
 Use Case Name: Quản lý tài khoản & phân quyền
 Actor: Admin
 Description: Tạo/sửa/xóa tài khoản và gán vai trò.
Precondition: Admin đã đăng nhập.
 Main Flow:
 1. Admin mở trang quản lý → tạo/sửa/xóa user → lưu thay đổi.
 Alternative / Exception Flow: Nếu thông tin thiếu → báo lỗi.
 Postcondition: User mới/được cập nhật trong DB.
